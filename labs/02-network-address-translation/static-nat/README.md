# Static NAT Lab

This lab demonstrates **Static Network Address Translation (Static NAT)** using Cisco Packet Tracer.

## Objective
- Configure a single router to translate inside private IPs to public IPs.

## Topology
- 1 Router
- 2 Switches
- 2 Servers (outside)
- 2 Laptops + 3 Desktops (inside)
